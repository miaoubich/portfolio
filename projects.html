Vendor management System Solution

The purpose of the Vendor Management System is to streamline and enhance the management of vendors, products, and orders within an organization. By leveraging microservices architecture using Java Spring Boot and Spring Cloud, the system ensures efficient handling of vendor-related operations such as profile and product management, order processing, and reporting. It facilitates seamless integration and automation of these processes, providing a scalable, secure, and user-friendly platform. This leads to improved vendor relationships, optimized product management, and efficient order fulfillment, ultimately driving operational efficiency and business growth.

Employee Management System

This is a straightforward EMS full-stack monolithic application that utilizes the Command Query Segregation Responsibility (CQRS) design pattern. The application comprises two services, each with its own DBMS. One service handles command APIs, while the other manages query APIs. Communication between these services follows an asynchronous pattern. The technologies employed in this project include Java 17, Spring Boot, MySQL, PostgreSQL, Apache Kafka, Docker, Jenkins, JUnit, MockMvc, and Postman.
